# Stardew-Valley-GUIDE

A friendly, beginner-focused guide to **Stardew Valley**, kept up to date with the
current game version (**1.6.15** / console **1.6.15.1**).

## Guides

| Guide | What it covers |
| --- | --- |
| [Relationships & Gifting Manual](RELATIONSHIPS.md) | How friendship works, the fastest way to max hearts, romance & marriage, universal gift tastes, and **per-character Loves / Likes / Dislikes** for every villager, plus a birthday calendar. |

_More sections (Farming, Fishing, Mining, Cooking, Community Center, Money-Making) are
planned — see the suggested structure in the project skill._

## About this project

- **Tone:** Beginner-friendly and spoiler-aware.
- **Format:** Markdown-first, readable directly on GitHub.
- **Accuracy:** Tips note the game version when behavior is version-specific.

This repository also keeps a small "memory" so help stays consistent across sessions:
the project skill lives in `.kiro/skills/stardew-valley-guide/SKILL.md`, and the
running log of decisions and chat history lives in [`AGENT.md`](AGENT.md).
